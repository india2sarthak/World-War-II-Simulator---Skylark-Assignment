# Technical Stack

Libraries used :-  

1.) MapBox (Android)
2.) Google Volley (Android)
3.) RecyclerView  (Android)
4.) LAMP Stack( code : https://github.com/india2sarthak/WWII-BackEnd- )(Backend)
